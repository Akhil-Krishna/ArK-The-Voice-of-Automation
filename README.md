# ArK : The Voice of Automation

ArK is a voice enabled personal assistant for a Developer or any user. It can simply answer any querries and also it can be connected to other devices and it will do the specific task the user wants to do on their own
Initially it will do simple tasks like calculation , write docs , txt files , pitch ideas , write code on the specified IDE etc

## Features

1. Question- Answering related to any topic
2. Voice enabled and can change it
3. Personalization
4. Hardware device
5. Coding , Debugging
6. Writing tasks

## Prototypes

1. ArK Bot 0.1.0 :- Speech recognition and simple bot task
2. ArK Bot 1.0.0 :- Ai powered question answering bot
